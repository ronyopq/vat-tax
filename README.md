# VAT & TAX Calculator (BD) — Dual Language, Animated, Responsive

Features:
- **Bangla/English toggle** (top-right) with instant UI switch.
- **Modern, mobile-first UI** with glassmorphism cards and animated background.
- **Responsive layout** (CSS Grid/Flex).
- **Playful animations** on *Calculate* (page ripple + confetti) and **count‑up results**.
- **Searchable dropdown** with Choices.js.
- **Font Awesome** colorful, animated icons.
- **One-click PDF download** of full-page screenshot with site name, URL, and current date.

Local preview:
```bash
python -m http.server -d site 8000
# open http://localhost:8000/
```
Deploy to GitHub Pages:
1) Upload the contents of `site/` to your repo root.
2) Settings → Pages → Source: Deploy from a branch → Branch: main (root) → Save.
